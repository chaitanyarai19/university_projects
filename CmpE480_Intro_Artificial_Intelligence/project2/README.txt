This program is written in Clips language. 

Prerequisites: 
"INPUTS.txt", "SYMPTOMS.txt" and "ILLNESS.txt" files must be on same directory with "medicalExpert.clp" file. Write following to the clips terminal to run the code:

> (load medicalExpert.clp)

It should return "TRUE", then type the following to get results and "OUTPUTS.txt" file.

> (runCode)

It should write on terminal one by one:
- Reading files...
- Creating dictionary for illnesses and assigning them to each user...
- Creating file named OUTPUTS.txt...
- Done!
That means, code run with 0 error and "OUTPUTS.txt" file is created under that directory.
